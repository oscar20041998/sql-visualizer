# Settings & Preferences ⚙️

Customize the SQL Visualizer to match your workflow and preferences.

## Theme Switching

- **Dark mode** (default) - Reduced eye strain during extended coding sessions
- **Light mode** - Traditional appearance with high contrast
- **Persistent settings** - Settings survive page refresh
- **Quick toggle** - Moon/Sun button in sidebar for instant switching

## Language Support

- **English (en)** - Complete UI translations
- **Vietnamese (vi)** - Full Vietnamese localization for all features
- **All UI text and labels translated** - Including JOIN Analysis terms
- **Quick toggle** - Globe button in sidebar

## Graph Visualization Options

### Layout Algorithms

| Algorithm          | Use Case                               | Best For                                            |
| ------------------ | -------------------------------------- | --------------------------------------------------- |
| **Dagre**          | Hierarchical layout for clear flow     | Complex queries with clear data flow dependencies   |
| **Force-Directed** | Organic spacing based on relationships | Balanced relationship viewing, intuitive clustering |
| **Grid**           | Structured grid arrangement            | Organized table arrangement, large flat queries     |

### Edge Style

- **Smooth Bezier:** Curved connections with smooth transitions
- **Straight:** Direct lines between nodes
- **Step:** Staircase pattern connections

## Auto-Analyze Setting

- **Enable:** Automatic analysis on paste/type
- **Disable:** Manual "Analyze" button required
- When enabled: Results update instantly as you type
- When disabled: Useful for pasting very large queries (avoids slowdowns)

## Sidebar State

- Collapsible sidebar for more screen space
- Collapse/expand toggle with keyboard shortcut
- Persistent state preference (remembered between sessions)

## Localization

- All UI elements, buttons, labels translated
- Feature descriptions in both languages
- Guidelines available in both English and Vietnamese
- Analysis results use selected language

## Best Practices

### Theme Selection

- **Choose dark theme** for reduced eye strain during extended coding sessions
- **Use light theme** in bright environments or for presentations
- **Switch frequently** if working multiple hours (reduce fatigue)

### Graph Layout Selection

- **Dagre layout** for hierarchical dependencies (clear flow of data)
- **Force-Directed** for balanced relationship viewing (intuitive clustering)
- **Grid** for structured, organized table arrangement
- **Test all three** to find your preference for different query types

### Auto-Analysis

- **Enable** if you edit queries frequently (real-time feedback)
- **Disable** for very large queries to avoid performance delays
- **Toggle based on task** - Enable for optimization work, disable for paste-and-analyze

### Sidebar

- **Collapse** for more screen real estate during visualization
- **Expand** for easy navigation between pages
- **Keyboard shortcut** for quick toggle while analyzing

## Accessibility

- High contrast options available in both themes
- Clear color distinctions for JOIN types and complexity levels
- Keyboard navigation support throughout the application
- Hover tooltips for additional context
